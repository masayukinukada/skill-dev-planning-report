# スキル分析・育成計画 レポートジェネレーター

IT部門のチームリーダーが、スキル分析・育成計画データをブラウザ上のフォームに入力し、整形されたHTMLレポートをワンクリックで生成・ダウンロードできるツール。

**サーバー不要・インストール不要。`index.html` をブラウザで開くだけで動作します。**

---

## 使い方

### 1. ファイルを開く

```
index.html をブラウザ（Chrome / Edge 最新版）でダブルクリック
```

### 2. フォームを入力する（4ステップ）

| ステップ | 入力内容 |
|---------|---------|
| Step 1 — 基本情報 | 年度・部門・リーダー名（必須）、対象期間・重点テーマなど |
| Step 2 — 予定案件 | 案件名・実施期間・スキル領域・重要度（複数件追加可） |
| Step 3 — 育成方針 | スキル領域別に目指す人材像・課題・育成方針（複数件追加可） |
| Step 4 — 育成目標 | メンバー別に育成対象スキル・現在Lv・目標Lv・施策（複数件追加可） |

### 3. レポートを生成・ダウンロードする

1. Step 4 の「**レポートを生成**」ボタンを押す
2. 画面下部にプレビューが表示される
3. 「**HTMLをダウンロード**」でファイルを保存
4. ダウンロードしたHTMLをブラウザで開いて確認・印刷・共有

---

## ファイル構成

```
skill-dev-planning-report/
├── index.html          # アプリ本体（HTML / CSS / JS をすべて内包）
├── README.md           # このファイル
├── CLAUDE.md           # Claude Code 向け実装指示書
├── SPEC.md             # 機能仕様書
├── ARCHITECTURE.md     # 設計詳細
├── REQUIREMENT.md      # 要件定義書
└── steering/           # タスクごとの設計決定記録
```

---

## 生成されるレポートの構成

| セクション | 内容 |
|-----------|------|
| 表紙 | 年度・チーム・リーダー・対象期間・重点テーマ・担当領域概要 |
| 1. 予定案件一覧 | 案件カード（名称・概要・実施期間・スキル領域・重要度バッジ） |
| 2. 育成方針 | スキル領域別カード（目指す人材像・課題・方針） |
| 3. メンバー別育成目標 | メンバーカード（アバター・Lvバー・育成施策） |
| 参考. スキルレベル定義 | Lv1〜5 の定義テーブル |
| フッター | チーム名・生成日時 |

## 技術仕様

| 項目 | 内容 |
|------|------|
| 構成 | HTMLファイル 1つ（単一ファイル完結） |
| 言語 | HTML / CSS / Vanilla JavaScript |
| 外部依存 | なし（CDN・npm・サーバー不要） |
| 動作環境 | Chrome / Edge 最新版（`file://` プロトコル対応） |
| セキュリティ | `escapeHTML()` によるXSS対策済み |
| 印刷 | `@media print` 対応（Ctrl+P で印刷可） |

---

## スキル領域の選択肢

案件・育成方針・育成目標で共通して使用する選択肢（20種類）：

- **オープンシステム**: H/W、OS、運用監視、セキュリティ、シンクライアント基盤、開発言語、自動化、データ分析
- **ネットワーク**: WAN/LAN環境、ネットワーク関連システム、セキュリティ、ネットワーク管理、ログ管理
- **コールセンター基盤**: 音声系システム、CTI基盤、アプリケーション基盤
- **クラウド推進**: システム戦略策定、クラウド基盤、AI基盤、アーキテクチャ策定

---

## 注意事項

- 入力データはブラウザを閉じると消えます。レポート生成後は必ずHTMLをダウンロードしてください。
- 機密情報を含む場合は、生成・ダウンロード後にブラウザの履歴を削除することを推奨します。
