# タスク4-4 — ヘルパー関数群

**作業日**: 2026-03-27
**対象ファイル**: `index.html`

---

## 実装内容

### priorityBadge(p)
- 引数: `'高'` / `'中'` / `'低'`
- 戻り値: 色分けされたバッジ HTML（高=赤系、中=オレンジ系、低=緑系）
- 未知の値は中扱い

### lvBar(lv)
- 引数: 数値 1〜5
- 戻り値: グリーン/グレーの5マス棒グラフ HTML ＋ `Lv{n}` テキスト
- 範囲外はクランプ（min:1, max:5）

### initials(name)
- 引数: 氏名文字列（例: `"山田 太郎"`）
- 戻り値: 最大2文字のイニシャル（姓名スペース区切り → 各1文字目、それ以外 → 先頭2文字）
- 空文字の場合は `'?'` を返す

### avatarColor(name)
- 引数: 氏名文字列
- 戻り値: 8色パレットから文字コード合計のハッシュで選んだ色コード
- 同じ氏名は常に同じ色になる

---

## 設計上の決定事項

| 決定内容 | 選択理由 |
|---------|---------|
| ヘルパーは生HTMLを返す | テンプレートリテラルに直接埋め込めて読みやすい |
| スタイルはインラインのみ | レポートHTML は独立ファイルになるため、外部CSSに依存できない |
| アバターカラーをハッシュで決定 | 毎回同じ色になるため視認性が安定する |
