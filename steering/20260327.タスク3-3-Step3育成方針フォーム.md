## 実装内容

### HTML
- 2カラムグリッド（`.field-grid`）に3フィールド配置:
  - `f-focus`: 重点スキル領域（テキスト）
  - `f-target`: 育成対象層（セレクト: 若手中心/中堅中心/次期リーダー候補/全層）
  - `f-vision`: 目指す人材像（テキスト）
- 全幅テキストエリア2つ:
  - `f-gap`: 現状の課題・ギャップ（4行）
  - `f-policy`: 育成方針（5行）
- 「← 戻る」/ 「次へ →」ボタン

### JS
- `next(2)` は既存の汎用実装で対応済み（Step3はすべて任意入力のためバリデーションなし）

## 設計上の決定事項

| 決定内容 | 選択理由 |
|---------|---------|
| すべて任意入力 | SPEC.md FR-04 の仕様通り。強制しない |
| テキストエリアを全幅配置 | 課題・方針は長文になりやすいため `.field-grid` の外に配置 |
| テキストエリアに `margin-bottom` をインラインで付与 | `.field-grid` 外なので `gap` が効かないため |
