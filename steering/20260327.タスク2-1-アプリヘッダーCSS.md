# タスク2-1 — アプリヘッダーHTML+CSS

**作業日**: 2026-03-27
**対象ファイル**: `index.html`

---

## 実装内容

- `.app-header`: グリーン（`#02993B`）背景、上28px/左右32px/下24px のパディング
- `.app-header::after`: 右上に半透明の円形装飾（`rgba(255,255,255,.08)`）でさりげないアクセント
- `.app-header-label`: 「IT人財育成ツール」という小さなバッジを半透明白枠で表示
- `h1`: font-size 1.35rem、font-weight 700、letter-spacing 0.02em
- `p`: font-size 0.82rem、opacity 0.88

---

## 設計上の決定事項

| 決定内容 | 選択理由 |
|---------|---------|
| グラデーションではなくグリーン単色 | テーマカラー変更に伴い、単色の方が会社サイトのトーンに合う |
| `::after` 疑似要素で装飾 | 画像・SVG不使用でビジュアルにアクセントを加える（外部依存なし） |
| `.app-header-label` バッジを追加 | ツールの用途をひと目で伝える補助ラベル。会社サイトのセクションラベル風デザインを踏襲 |
| `overflow: hidden` を設定 | `::after` の円がはみ出さないようにクリッピング |
