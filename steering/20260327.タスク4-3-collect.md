---

## 実装内容

### escapeHTML(str)
- `&`, `<`, `>`, `"`, `'` の5文字をHTMLエンティティに変換
- レポート生成時にすべての値を必ずこの関数に通してから埋め込む

### val(id)
- `document.getElementById(id).value.trim()` の短縮ヘルパー
- 要素が存在しない場合は空文字を返す

### collect()
戻り値オブジェクト構造:
```javascript
{
  basic:    { year, team, leader, period, date, theme, scope },
  projects: [{ name, overview, start, end, skill, priority }],  // name が空の行は除外
  policy:   { focus, vision, gap, policy },
  trainees: [{ name, measure }],  // name と measure が両方空の行は除外
  members:  [{ name, skill, currentLevel, goalLevel, measure }],  // name が空の行は除外
}
```

---

## 設計上の決定事項

| 決定内容 | 選択理由 |
|---------|---------|
| `collect()` は生値（未エスケープ）を返す | エスケープはレポート生成時のみ適用。二重エスケープ防止 |
| `currentLevel` / `goalLevel` を `parseInt()` で数値化 | `lvBar()` など数値前提のヘルパーに渡すため |
| 空行フィルタリングの基準を `name` のみにする（案件・メンバー） | 必須フィールドが埋まっていない行をレポートから除外 |
