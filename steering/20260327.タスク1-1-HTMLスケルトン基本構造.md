# タスク1-1 — HTMLドキュメント基本構造

**作業日**: 2026-03-27
**対象ファイル**: `index.html`

---

## 実装内容

- `<!DOCTYPE html>` / `<html lang="ja">` / `<head>` / `<body>` の基本骨格を作成
- `<meta charset="UTF-8">` を最初のメタタグとして配置（日本語文字化け防止）
- `<meta name="viewport">` でモバイル表示にも対応
- `<style>` タグ：アプリUI用CSS（カラーパレットコメント + 基本リセット + `.app` ラッパー）
- `<script>` タグ：JS記述エリアを確保（中身は後続タスクで追記）
- 各セクションにコメントプレースホルダーを配置

---

## 設計上の決定事項

| 決定内容 | 選択理由 |
|---------|---------|
| `<style>` / `<script>` を1つにまとめる | 単一ファイル構成の維持。外部ファイル分離は不可（CLAUDE.md制約） |
| `hidden` 属性によるステップ表示切替 | CSSの `display:none` との二重管理を避ける。JSから `removeAttribute('hidden')` / `setAttribute('hidden','')` で制御 |
| カラーパレットをコメントで冒頭定義 | CLAUDE.mdのCSS規約に従い、定数として冒頭にまとめる |
| `.app { max-width: 860px }` | ARCHITECTURE.mdの設計（最大幅860px中央揃え）に準拠 |
