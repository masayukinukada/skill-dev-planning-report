**作業日**: 2026-03-27
**対象ファイル**: `index.html`

## 実装内容

- `.step-nav`: `position: sticky; top: 0` でスクロール時も追従（z-index: 100）
- `.step-btn`: 縦並びレイアウト（数字バッジ + ラベル文字）、`transition` でなめらかな状態変化
- `.step-num`: 24×24px の円形バッジ（状態で色が変わる）
- 3状態のスタイル定義:
  - **未着手**: グレー円（`#e5e7eb`）、グレー文字（`#9ca3af`）
  - **active（現在）**: グリーン円（`#02993B`）、グリーン下ボーダー、白文字
  - **done（完了）**: オレンジ円（`#F39346`）、数字を隠して `✓` を `::before` で表示

---

## 設計上の決定事項

| 決定内容 | 選択理由 |
|---------|---------|
| `position: sticky` を採用 | 長いフォームをスクロールしても現在ステップが常に見える |
| 数字バッジを縦並びに変更 | 横並びより視認性が高く、会社サイトのUIトーンに合う |
| `done` 時の数字を `color: transparent` で隠す | 数字テキストは残しつつ `::before` で `✓` を重ねる。JSで数字を書き換えない設計 |
| `transition: color .2s` | タブ切り替えのアニメーションをなめらかにする |
| `:hover` スタイルを追加 | クリッカブルであることをユーザーに伝える |
